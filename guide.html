Tool 01
🪧
會員優惠海報系統
上傳操盤 PDF，自動生成 A4 列印海報
開啟工具 →
操作步驟
  1. 1點「開啟工具」，頁面會自動帶入門市名稱
  2. 2點上方「選擇操盤 PDF」，上傳當期活動操盤檔
  3. 3系統自動解析活動內容，生成 iPhone／Mac／iPad／Watch 四版海報預覽
  4. 4點上方主題按鈕可切換 4 種海報風格
  5. 5任何文字可直接點擊編輯，調整為門市實際活動內容
  6. 6點右上「列印」，用 Safari 選「另存 PDF」或直接紙本印出（A4）
適用時機
每個活動期換期時，快速製作 4 版全品項海報，貼於門市看板
日期自動偵測
上傳的 PDF 檔名若含日期格式（如 0519-0531操盤.pdf），活動期間自動填入
注意事項
請使用 Safari 列印，Chrome 分頁效果可能不正確
Tool 02
📦
到貨提醒儀表板
管理客訂待通知清單，三色狀態追蹤
開啟工具 →
操作步驟
  1. 1從 POS 系統匯出客訂 Excel（.xlsx）
  2. 2拖曳或點擊上傳至工具
  3. 3系統分 Apple 正廠 / 副廠配件 兩個分頁顯示所有待通知訂單
  4. 4查看三色狀態徽章，掌握通知優先順序
  5. 5通知完畢後點擊徽章切換為「已通知」，記錄進度
三色狀態說明
🔴 緊急 🟡 一般 ✅ 已通知
適用時機
每日開門前或到貨後，快速確認哪些訂單尚未聯繫客人
注意事項
資料不會自動儲存，每次刷新需重新上傳 Excel
Tool 03
📊
配件庫存管理系統
庫存水位分析 + 補貨建議 + 補貨發信
開啟工具 →
操作步驟
  1. 1從 POS 匯出 副廠庫存.xls 與 銷售 sa.xls（同時段）
  2. 2兩份檔案拖入工具上傳區（可同時上傳）
  3. 3系統根據銷售速度計算預計耗盡天數，分四色水位顯示
  4. 4勾選需要補貨的品項
  5. 5點「補貨發信」生成補貨表格，可複製後寄 Email 或匯出 .ics 行事曆提醒
四色水位說明
🔴 緊急 <14天 🟠 建議 14–30天
🟡 注意 30–60天 🟢 充足 >60天
適用時機
每月定期檢視，或促銷前確認庫存是否充足
自動排除
品名含「燦坤」「保護貼貼服務」「貼服務」的品項自動不顯示
Tool 04
🛒
配件陳列建議
銷售 TOP 15 × 花車陳列對照 × 改善建議
開啟工具 →
操作步驟
  1. 1開啟工具，輸入密碼登入（tk3c027）
  2. 2切換到「更新資料」頁籤
  3. 3拖入 POS 銷售 Excel（sa.xls），系統自動辨識欄位
  4. 4確認欄位對應無誤後，點「確認匯入」
  5. 5切換到「花車陳列」，更新各品項的陳列狀態（有陳列 / 未陳列 / 位置不佳）
  6. 6回到「更新資料」點「儲存並套用」，完成更新
五個功能頁籤
銷售排行 TOP 15、圖表分析、花車陳列對照、建議行動、更新資料
報告期間
上傳 Excel 後系統自動偵測最早與最晚交易日期，填入報告期間
資料儲存位置
存於瀏覽器 localStorage，不同裝置或清除瀏覽器資料後需重新匯入
Tool 05
📈
配件陳列分析系統
自動排名 TOP 15 + 貨架示意圖 + 導購話術
開啟工具 →
操作步驟
  1. 1拖入 POS 銷售 Excel（sa.xls）至上傳區
  2. 2系統自動篩選配件類別（031xxxxx），計算 TOP 15 銷售排名
  3. 3查看「貨架示意圖」，了解建議陳列區位（A 黃金區 / B 次要區 / C 一般區 / D 底部）
  4. 4點「多期比較」，可同時上傳多份 Excel 對照不同月份銷售變化
  5. 5查看「導購話術」，取得各品類的開場問句與需求引導框架
與 Tool 04 的差異
Tool 04 記錄花車實際陳列狀態，需手動維護
Tool 05 純分析工具，每次上傳即重新計算，不需登入
品類自動辨識
充電線、充電器、行動電源、轉接器、iPad 筆等品類由系統自動分類，無需手動標記
適用時機
每月銷售結算後，快速取得當期排名與陳列建議
Tool 06
🎓
小教室簡報生成器
勾選主題與對象，自動生成顧客小教室簡報、宣傳海報與社群文案
開啟工具 →
操作步驟
  1. 1第一次使用先點右上「⚙️ 門市設定」，填入門市名稱與報名方式（每台電腦設定一次）
  2. 2左側勾選要講的主題（可多選）、右側選對象年齡層
  3. 3點「生成簡報」，可線上點字編輯、「列印／PDF」或「下載 .pptx」用 Keynote 開啟
  4. 4點「📣 宣傳海報」依當次主題產生海報，可切 A4／社群方形與四種配色，存圖或列印
  5. 5點「✍️ 社群文案」一鍵複製開課貼文到 LINE@／社群
  6. 6找不到主題？用「🔗 用官網連結產生主題」貼 Apple 官網連結，依指示用 AI 補上新主題
適用時機
舉辦顧客小教室／體驗講座時，快速備好簡報、開課海報與社群宣傳文案
多選主題會融合
選一個以上主題時，會依共同主軸智慧融合成一場主題式講座（固定約 11 頁），不是逐一產品分頁
注意事項
門市設定、海報配色等偏好存在該台電腦的瀏覽器，換電腦需重新設定一次
